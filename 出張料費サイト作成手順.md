# 出張料費精算アプリ用サイトの作成手順

## デプロイ後のURL

**https://shucchou-ryohi-seisan--omake-app.web.app/**

（shucchou-ryohi-seisan = 出張料費精算）

---

## 手順（ターミナルで実行）

### 1. サイトを作成

```bash
cd ~/Desktop/FKT-SRK-APP
firebase hosting:sites:create shucchou-ryohi-seisan
```

※初回のみ。サイトが既に存在する場合はスキップして次へ。

### 2. デプロイ

```bash
firebase deploy --only hosting
```

---

## これで出張料費精算アプリ専用のURLになります

- **omake-app.web.app** → デフォルトサイト（他のアプリ用）
- **shucchou-ryohi-seisan--omake-app.web.app** → 出張料費精算アプリ専用

他のアプリも同様に、それぞれ別のサイトIDで作成すれば、専用URLで分けて管理できます。
