# 出張料費管理アプリ デプロイ手順

**https://shucchou-ryohi-kanri--omake-app.web.app/**

---

## 初回のみ：Firebaseサイトの作成

1. [Firebase Console](https://console.firebase.google.com/) を開く
2. プロジェクト「omake-app」を選択
3. **Hosting** → **追加サイト** をクリック
4. サイトIDに `shucchou-ryohi-kanri` を入力して作成

---

## デプロイ実行

```bash
cd ~/Desktop/FKT-SRK-APP

# 1. GitHub にプッシュ
git add firebase.json
git commit -m "出張料費管理のURLでデプロイするようサイト名を指定"
git push

# 2. Firebase にデプロイ
firebase deploy --only hosting
```
